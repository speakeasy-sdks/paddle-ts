# SchemaStatusTransaction

Status of this transaction. You may set a transaction to `billed` or `canceled`, other statuses are set automatically by Paddle.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `Draft`     | draft       |
| `Ready`     | ready       |
| `Billed`    | billed      |
| `Completed` | completed   |
| `Canceled`  | canceled    |
| `PastDue`   | past_due    |
