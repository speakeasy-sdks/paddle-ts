# AdjustmentCreateType

Type of adjustment for this transaction item. `tax` and `proration` are automatically created by Paddle. 
Include `amount` when creating a `partial` adjustment.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `Full`      | full        |
| `Partial`   | partial     |
| `Tax`       | tax         |
| `Proration` | proration   |
