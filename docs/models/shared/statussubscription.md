# StatusSubscription

Return entities that match the specified status. Use a comma separated list to specify multiple status values.


## Values

| Name       | Value      |
| ---------- | ---------- |
| `Active`   | active     |
| `Canceled` | canceled   |
| `PastDue`  | past_due   |
| `Paused`   | paused     |
| `Trialing` | trialing   |
